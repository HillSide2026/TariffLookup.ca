# Skill: Source Package Audit

Purpose:
- Audit official-source EU packages before normalization decisions are made.

Use when:
- reviewing raw EU source artifacts
- validating whether a row is safe to normalize

Required behavior:
- inspect branch structure and measure consistency
- identify diverging duty outcomes
- record whether the source supports normalization, blockage, or manual review

Outputs:
- source audit note
- normalization readiness assessment
- unresolved-source warning
