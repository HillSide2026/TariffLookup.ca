# Skill: Classifier Profile Authoring

Purpose:
- Add or revise description-to-HS classification profiles safely.

Use when:
- editing `classification-service.ts`
- introducing new positive patterns or exclusions

Required behavior:
- keep profiles narrow and evidence-backed
- encode positive, required, and excluded terms deliberately
- prefer precision over broad recall

Outputs:
- profile change
- rationale note
- supporting regression cases
