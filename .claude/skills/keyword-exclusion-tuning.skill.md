# Skill: Keyword And Exclusion Tuning

Purpose:
- Tune inclusion and exclusion logic to reduce false matches.

Use when:
- a classifier is colliding with adjacent product types
- a normalized row needs stricter routing

Required behavior:
- inspect competing matches
- harden exclusion logic for nearby product classes
- avoid broad keywords that drag in unrelated goods

Outputs:
- keyword adjustment
- exclusion adjustment
- conflict-reduction note
