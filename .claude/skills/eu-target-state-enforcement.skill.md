# Skill: EU Target State Enforcement

Purpose:
- Enforce the three allowed end states for each EU group: `normalized`, `blocked-with-guidance`, or `manual-review`.

Use when:
- adding rows
- changing guidance
- triaging unresolved codes

Required behavior:
- place every active-catalog group in exactly one state
- prefer `normalized` only when safe
- never hide ambiguity behind a false normalized row

Outputs:
- state decision
- state-transition note
- rule-consistency check
