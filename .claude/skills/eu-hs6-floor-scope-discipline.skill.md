# Skill: EU HS-6 Floor Scope Discipline

Purpose:
- Keep work aligned to the EU-only stage and the HS-6 floor of about `5,000` groups.

Use when:
- a change risks drifting into non-EU scope
- a task confuses current prototype rows with the real completion floor

Required behavior:
- prioritize EU only
- treat `~5,000` HS-6 groups as the floor, not the current dataset size
- avoid scope expansion into other jurisdictions unless ML1 changes scope

Outputs:
- scope note
- out-of-scope flag
- narrowed task framing
