# Skill: Coverage Gap Quantification

Purpose:
- Quantify the gap between the current EU baseline and the HS-6 floor target.

Use when:
- reporting progress
- prioritizing where to add the next rows or states

Required behavior:
- distinguish normalized-row count from total catalog count
- make gaps explicit and numerically grounded
- avoid treating current prototype rows as the real completion floor

Outputs:
- gap count
- progress snapshot
- target-state delta note
